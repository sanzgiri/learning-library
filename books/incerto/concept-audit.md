# Incerto Concept Audit

Checklist against Taleb's own [Incerto Glossary](https://www.fooledbyrandomness.com/glossary.pdf) (read in full, not search-summarized) plus terms confirmed via other primary sources for the later books (*Antifragile*, *Skin in the Game*). Retroactive pass, run after the fact — this is exactly the step that caught ergodicity being missing on the first pass through this course; running it properly the first time on a future book should catch equivalents before lessons are written, not after.
